{% tab title="3️⃣Logistics" %}
* Ability to distribute the suppliers and customers beginning balances into separated previous invoices.

Instead of displaying the opening balance as a consolidated amount, it can be broken down into separate invoices representing the balance amount. This enables direct payments/ receivables to be made against each invoice.\
The payment/ collection process has also been updated to target these detailed invoices rather than the aggregated opening balance. This improvement enhances transparency and ensures more precise accounting tracking and reconciliation

_**2)**_**&#x20;Monitoring and operation of concrete products.**

Allow creating a delivery plan from the Sales Order screen:\
The ability to create a delivery plan directly from the Sales Order screen has been added ‒ button “+” ‒to simplify the process of scheduling order deliveries. This option enables the user to prepare a plan that includes all delivery details, including pump information, without the need to navigate to another screen.

Add a link between mixers and pumps:\
Each mixer is now linked to a single pump, which is assumed to pump the full quantity carried by the mixer.

Add a link between the vehicle and the default driver (one-to-one) and apply it when adding mixers and pumps:\
A direct one-to-one relationship between each vehicle and its default driver has been implemented. This relationship is applied automatically when adding mixers and pumps.

{% tab title="5️⃣Real Estate Investment and Management d" %}
**Added a Sales Representative filter to the&#x20;**_**Item Profitability by Groups**_**&#x20;report**

A new option has been added in the settings called "Allow using the same phonebook to multiple customers", with its default value set to disabled. This option cannot be deactivated if there are already multiple customers saved with the same phone directory.

When enabled, all phone directory records will be displayed in the lookup list within the Customers screen, and the same phone directory can be linked to more than one customer.

When disabled, the phone directory field in the Customers screen will function as it currently does, without allowing the same directory to be assigned to more than one customer.

Additionally, the phone application settings have been updated to allow reading all units associated with all customers linked to the same phone number.
