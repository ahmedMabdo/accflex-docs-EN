# Technical Setup Guide

#### E-Mail Configuration

Enter the E-Mail setting information:

1. Select **E-Mail service provider** (SMTP, or SendGrid).
2. Enter the **SMTP Host**.
3. Enter **Port number**.
4. Enter **User E-mail Address**, and **Password**.
5. Enter **Sender E-mail**.
6.  Enter **Sender Name**.<br>

#### SMS Configuration

Reach out to your service provider, to subscribe to the service, and get a SMS URL and a user code.

1. Enter the **SMS URL**.
2. Enter **SMS User Code**.
3. Enter the **Password**.
4. Enter the **Sender Name**.
5. Enter the **SMS Global Country Code**.
6.  Select the **SMS Provider Type**.<br>
