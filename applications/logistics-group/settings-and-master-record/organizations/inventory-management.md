# Inventory Management

The screen is designed to set up and define warehouses, whether they are intended for inventory storage or manufacturing purposes. The user starts by entering the warehouse name in both primary and secondary languages. Following this, they specify the main branch associated with the warehouse and select the applicable calendar. Next, the user defines the warehouse's location by selecting the country, city, and district. Finally, they assign the scrap account linked to the warehouse and specify the corresponding cost center for accounting purposes. This comprehensive setup ensures all necessary details are captured for proper warehouse management.
