#### Link a User to a Branch&#x20;

When you link a user to a specific branch, this user will not be able to manage any other branches transaction.&#x20;

#### Security Level

Set the security level for this user account:\
\- Highest Security Level (5); this account is protected, it's data can only be viewed by accounts having the same security level. Furthermore, this account is allowed to view other accounts data, whether they have the same security level 5 or less. \
\
\- Lowest Security Level (1); this account data is accessible by all accounts, and can only view data of accounts having the same security level 1. This account can not view the higher security accounts data.

#### User Permissions

The program displays a table of all the modules permissions. Permissions are detailed in; Add, Edit, Delete, View, and Print. You can set detailed permissions for each user, and special permissions for some modules.&#x20;

You can rearrange the displayed table of module categories, and set which categories to be shown. Click on the filter icon, checkmark your selections, then **Close**.&#x20;
