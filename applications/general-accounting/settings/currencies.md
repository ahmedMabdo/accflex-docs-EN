#### Add a New Currency Factor

When you edit or add a new currency factor, the new factor will be applied for all the upcoming transactions, starting from the update date.\
To view a registered currency information, or to add a new currency factor:

* Click on **Open** ![](../../../.gitbook/assets/open.png)/ **Choose Currency** ![](<../../../.gitbook/assets/GA web En 032.png>).
* Select the assigned currency, to view or edit.
