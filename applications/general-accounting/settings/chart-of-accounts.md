#### Add a New Account ![](../../../.gitbook/assets/New.png)

To add a new main account, enter the account data, then **Save**. You can add a new sub account by:&#x20;

* Click on **New** ![](<../../../.gitbook/assets/image (156).png>), enter the name of the sub account, and select a main account for it. Account number, type, and nature are automatically applied according to the main account.
* Select a main account, click **New** then enter the name of the sub account, and according to the main account, the sub account number, type, and nature are automatically applied.&#x20;

#### Import Accounts from Excel ![](<../../../.gitbook/assets/import from excel.png>)

Registering sub accounts is never easier. If you have the sub accounts listed in an excel sheet, by simple steps you can import these accounts from excel:

* Select the main account, then click on **Open**.
* Click on Import from Excel icon, then follow instructions.
* Copy the table of sub accounts from the excel file, without the column headers, also, cells has to be arranged as the shown, then click **Paste**.
* All copied cells will be displayed, and easy to edit. Then click **Import**.&#x20;

<figure><img src="../../../.gitbook/assets/GA 046.png" alt=""><figcaption><p>Import sub accounts from Excel</p></figcaption></figure>

#### Move Accounts ![](<../../../.gitbook/assets/Move accounts.png>)

The program enables you to move sub accounts between main accounts. If a sub account was mistakenly set to a wrong main account, instead of deleting and adding a new account, move the account manually to the right place.

#### Adjust Accounts ![](<../../../.gitbook/assets/Adjust accounts.png>)

After adding, editing and moving accounts, use the tool Adjust Accounts so the program automatically resets the numbers of the accounts.

#### Currency

General Ledger program gives you different currency options, to set for each new account. Checking the **Currency Factor Revaluation**, means that this account can re-evaluate the factor of its assigned currency. The program will re-evaluate currency factor, for all accounts that has this option checked, at the reconciliation of currency margin profit account.

<figure><img src="../../../.gitbook/assets/GA 047.png" alt=""><figcaption><p>Currency</p></figcaption></figure>

#### Account Branches

If an account is linked to specific branches, transactions on other branches cannot be made by this account.&#x20;

#### Accounts and Cost Centers

To monitor and follow up with the expenses & revenues of each account, and profit & loss of cost centers.

#### Security Level

AccFlex ERP programs offer a data security system:\
\- Highest Security Level (5); this account is protected, it's data can only be viewed by accounts having the same security level. Furthermore, this account is allowed to view other accounts data, whether they have the same security level 5 or less. \
\
\- Lowest Security Level (1); this account data is accessible by all accounts, and can only view data of accounts having the same security level 1. This account can not view the higher security accounts data.

#### Transfer Transactions ![](<../../../.gitbook/assets/image (27).png>)

The program enables you to transfer some or all saved transactions from an account to another, by a few simple steps. A detailed list of the transactions is shown, you can choose which transactions to transfer. To transfer transactions from an account to another:

* Choose the exporting/ source cost account, then click on **Transfer Transactions** icon, and select the destination/ importing account.
* Filter the transactions search results, by entering some data: start and end of the search period, application/ program, currency, and branch then click on **Preview**.
