Instead of manually entering balances data for each account, the General Ledger program allows you to easily import data of opening balances from Excel. Enter the required data: **Accounting Period, Branches, and Currency**, then click on the **Excel icon** ![](<../../../.gitbook/assets/open excel.png>).

* Select an Excel file.
* Select the Excel sheet, for the program to extract data from.
* Required columns headers are shown, match with headers in the file.
* Click on **Load Data**.
