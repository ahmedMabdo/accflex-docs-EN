# Journals Entries Reports

The program enables extracting the journals entries reports, in three different types:

* [Detailed Journals Report](journals-entries-reports.md#detailed-journals-report)
* [Summarized Journals Report](journals-entries-reports.md#summarized-journal-report)
* [Journals Period Report](journals-entries-reports.md#journal-period-report)

The **Detailed Journals Report** displays the basic information of each journal entry: numbers of journal entry, and archive, date, currency, journal entry description, value of journal (debit, or credit), also, journal type, currency factor, name and number of account, name and number of cost center, branch and the program used to create this journal entry.

The program allows four search results filtration methods, choose the suitable method to search and extract journal entries reports:

* [By Serial/ Date](journals-entries-reports.md#detailed-journal-report-by-serial-by-date)
* [By Account ](journals-entries-reports.md#detailed-journals-report-by-journal-type)
* [By User](journals-entries-reports.md#detailed-journals-report-by-user)
* [By Type](journals-entries-reports.md#detailed-journals-report-by-journal-type)

The Journal Period Report displays the basic data: name and number of accounts, sum of all journal entries of each account (debit or credit). Filter the search results by entering the required data: branches, the start and end of a period of time, and the number of accounts level then click **Preview**/ **Print**.&#x20;

{% hint style="info" %}
Level of accounts, is the number of figures of the accounts number, for example: account no. 11 is a level 2, account no. 111 is a level 3.&#x20;
{% endhint %}
