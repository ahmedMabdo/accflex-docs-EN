# Fixed Assets Actions

The General Ledger program enables classifying the fixed assets into main types, specify the depreciation for each type, and specify the value of depreciation for each type of assets.

Different assets screens of different uses are provided by  the program:

### Assets Types

The program enables sectioning fixed assets into main types, specify the depreciation method for each type, and the value of the depreciation specified for this type of asset. When a new asset is registered, the program automatically applies the settings of the asset type, depreciation percentage, and accounts registered to the new asset.

#### Add a new Assets Type ![](../../../.gitbook/assets/New.png)

To register a new assets type, enter data: assets type code, and name, depreciation method, and rate. Also, enter the accounts data related to this assets type: assets account, accumulated depreciation account, depreciation expenses account, dedicated depreciation account, sale profit account, sale loss account, purchase account, then click **Save** ![](../../../.gitbook/assets/Save.png).

Add all the accounts of the assets type, through the [Chart of Accounts](../settings/chart-of-accounts.md)&#x20;

#### Actions on Assets

The program enables applying different actions on assets: depreciate, reduce, renew, sell, and roll out. Select the assigned asset, then click **Edit**.

#### &#x20;Depreciate Asset

The program annually depreciate assets with the registered depreciation value, after depreciating assets, and transfer balances, at [accounting period](../settings/accounting-periods.md) locking. Also, you can manually depreciate assets at any time. Select the assigned asset, enter data, then click **Depreciate**.
