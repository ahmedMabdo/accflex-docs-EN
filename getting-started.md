#### Basic Info

Enter the basic information of your company: Name, Telephone, Fax, Email, Address, and Company logo. These information will always be shown on the header of every report, sheet and bill.

#### E-Invoice Settings

Enter the electronic invoice information, to apply the E-Invoice, according to the directions of the Egyptian TAX Authority. E-Invoice API Base URL, E-Invoice Client ID, E-Invoice Client Password, E-Invoice Identity URL, activity Type Code, Token Pin, E-Invoice Issuer Type (P: personal, B: business,  F: Foreign), Client ID, and Password.

#### Embedded Data

This feature enables sending notifications (SMS, or E-Mail), to a lot of contacts, just by one click. When you choose (Mobile, E-Mail) from the embedded data, the program searches through every account registered with mobile or E-mail, and send them the scheduled notifications. To use this feature:&#x20;

From Embedded Data, choose mobile for SMS, and Email address for email.

{% tabs %}
{% tab title="Installed Program" %}
<figure><img src=".gitbook/assets/main 014 (1).png" alt=""><figcaption><p>Choose mobile for SMS, and E-mail address for E-mail</p></figcaption></figure>
{% endtab %}

{% tab title="Online Program" %}
<figure><img src=".gitbook/assets/Main web En 09.png" alt=""><figcaption><p>Choose mobile for SMS, and E-mail address for e-mail</p></figcaption></figure>
{% endtab %}

Use the Embedded Data, to include specific and updated data, in some types of the sent notifications. For example; in the Welcome Customer, you can add the clients name and discount account ID. Also, in Payment Reminder, if you add the due amount, the program will search through the data of every client, and embed these data in the notification.
